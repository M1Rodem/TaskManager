Task Manager с логированием

О проекте
Консольное приложение на C# для управления списком задач с полноценным логированием и трассировкой операций. Проект разработан в рамках учебного задания по программированию.

Функциональные возможности
Добавление задач - ввод названия задачи, валидация пустых значений
Удаление задач - удаление по названию, проверка существования задачи
Вывод списка задач - нумерованный список всех задач
Логирование операций - запись всех действий в логи
Трассировка - пошаговое отслеживание выполнения операций

Установка и запуск
// Клонирование репозитория
git clone https://github.com/M1Rodem/TaskManager.git

// Переход в папку проекта
cd TaskManager/TaskManager

// Сборка проекта
dotnet build

// Запуск приложения
dotnet run

Использование
При запуске отображается меню с номерами действий:
=== TASK MANAGER ===

Меню:
1. Добавить задачу
2. Удалить задачу
3. Показать список задач
4. Выход
Выберите действие (1-4):

Пример работы:
Выберите 1 для добавления задачи
Выберите 3 для просмотра списка задач
Выберите 2 для удаления задачи
Выберите 4 для выхода

Логирование
Приложение создает два файла:
taskmanager.log - основные логи операций
tasks.txt - текстовый журнал всех действий

Уровни логирования:
INFO - информационные сообщения (успешные операции)
WARN - предупреждения (некритичные ошибки)
ERROR - ошибки (операции не выполнены)
TRACE - трассировка (пошаговое выполнение)

Особенности реализации
Использование пространства имен System.Diagnostics для логирования
Разделение логики на отдельные классы (Task, TaskManager, Logger)
Поддержка всех уровней логирования из задания
Детальная трассировка каждой операции
Валидация вводимых данных
